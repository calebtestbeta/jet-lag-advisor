## Why

專案目前缺少說明文件，新訪客無法快速了解這是什麼應用程式、如何使用、以及如何在本地端執行。加入 README 是開放原始碼專案的基本配備。

## What Changes

- 新增 `README.md` 於專案根目錄
- 內容涵蓋：專案簡介、功能列表、使用方式、本地端開發啟動步驟、技術棧說明

## Capabilities

### New Capabilities

- `project-readme`: 專案根目錄 README.md，提供簡介、功能說明、使用方式及開發指引

### Modified Capabilities

（無）

## Impact

- 僅新增 `README.md` 檔案，不影響任何現有程式碼或功能
