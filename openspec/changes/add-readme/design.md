## Context

專案是單一 `index.html` 的 PWA，無 build step，直接用 HTTP server 或 GitHub Pages 提供服務。

## Goals / Non-Goals

**Goals:**
- 新增 `README.md`，讓訪客在 GitHub 首頁就能了解專案

**Non-Goals:**
- 不建立多語言文件
- 不加入 API 文件或詳細開發指南

## Decisions

- 使用純 Markdown，不加 emoji，保持簡潔
- 語言：中文為主，專有名詞保留英文
- 章節：簡介 / 功能 / 使用方式 / 本地開發 / 技術棧

## Risks / Trade-offs

- 無技術風險，純文件新增
